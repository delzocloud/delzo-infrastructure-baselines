El formato sigue [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/) y el
versionado sigue [Semantic Versioning](https://semver.org/lang/es/).

### Qué significa cada número (SemVer aplicado a la guía)

- **MAJOR** (`2.0.0`) — cambia o revierte una recomendación existente; algo que antes se
  aconsejaba ahora se desaconseja (guía "breaking").
- **MINOR** (`1.1.0`) — se agrega una sección o recomendación nueva sin contradecir lo anterior.
- **PATCH** (`1.0.1`) — correcciones de redacción, typos, ejemplos o traducciones, sin cambiar
  el contenido de fondo.

## [1.0.0] - 2026-07-10

### Added
- Primera versión pública de las baselines de infraestructura (AWS · Azure · GCP).
- Diez secciones: separación de cuentas, direccionamiento IP/CIDR, DNS centralizado, naming,
  SSO/identidad, control de código, gestor de contraseñas, sistema de tickets, diagrama de
  integración y componentes complementarios.
- Publicación en tres idiomas: español (`es/`), inglés (`en/`) y portugués (`pt/`).
- Disclaimer en cada documento: es una guía de ejemplo, debe adaptarse a cada infraestructura.
